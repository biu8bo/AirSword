# 阶段 3：手势追踪 + 鼠标仿真

> 本阶段在 21 关键点基础上实现手势状态机、平滑追踪、死区、鼠标仿真，让"剑指"真正能当鼠标用。

## 阶段目标

实现 `LingXuZhi.Core` 的手势识别、平滑追踪、坐标映射，以及 `LingXuZhi.Platform.Mouse` 的鼠标控制，完成端到端"剑指当鼠标"功能。

## 前置约束（必须遵守）

1. **核心逻辑在 Core**：手势状态机、平滑滤波、死区、坐标映射全部在 `LingXuZhi.Core`，零平台依赖。鼠标控制在 `Platform.Mouse`，Windows SendInput 实现。
2. **接口解耦**：`IGestureRecognizer`（输入 21 关键点 → 输出手势状态）定义在 Core；`IMouseController`（移动/左键/右键/滚轮）定义在 Platform.Mouse。
3. **平滑优先**：鼠标移动必须平滑缓慢，禁止抖动。默认启用 EMA 滤波（系数可配），预留卡尔曼接口。
4. **死区强制**：手指在屏幕正中央死区（半径可配）内停止时，不产生移动事件。
5. **本阶段不做**：多手支持、惯用手切换、自定义手势、MediaPipe 切换（接口已预留即可）。

## 手势定义

基于 21 关键点判定（关键点编号见阶段 2 骨架拓扑）：

| 手势 | 判定条件 | 触发动作 |
|------|----------|----------|
| **剑指（移动）** | 食指(8)与中指(12)伸直并拢，无名指(16)/小指(20)弯曲，拇指(4)收拢，鼠标指针移动检测的采样点在食指和中指指尖的中心位置 | 移动鼠标指针 |
| **食指-拇指捏合（左键）** | 食指尖(8)与拇指尖(4)距离 < 阈值，中指(12)伸直 | 鼠标左键点击 |
| **食指-中指捏合（右键）** | 食指尖(8)与中指尖(12)距离 < 阈值（两指均伸直但靠拢） | 鼠标右键点击 |
| **手掌张开（滚轮）** | 五指全部伸直张开 | 上下摆动 → 滚轮滚动 |
| **空闲** | 以上均不满足 | 不动作 |

### 手指伸直/弯曲判定（建议）

- 伸直：指尖到掌根(0)的距离 > 该指根关节到掌根距离的 1.5 倍（或指尖 y 坐标超过指根关节，考虑手部朝向）。
- 弯曲：指尖在指根关节之下（朝掌心方向）。
- 具体阈值通过配置可调，先实现一套合理默认值。

### 捏合距离阈值

- 归一化坐标下的欧氏距离，默认阈值 0.05（约手掌宽度的 5%），可配。

### 滚轮方向判定

- 手掌张开状态下，追踪手腕(0)或手掌中心(9)的纵向位移。
- 向上摆动 → 滚轮向上，向下摆动 → 滚轮向下。
- 用位移变化率（速度）触发，避免持续滚动；每次触发后需回中才再次触发（状态机防抖）。

## 状态机

```
                 [张开检测到]
[空闲] ──剑指──> [移动中] ──捏合(食+拇)──> [左键触发] ──释放──> [移动中]
   │               │                                                       │
   │               └──捏合(食+中)──> [右键触发] ──释放──> [移动中]          │
   │                                                                       │
   └──手掌张开──> [滚轮模式] ──回中/握拳──> [空闲] <───────────────────────┘
```

- 状态切换需**去抖**（连续 N 帧 / T ms 满足条件才切换），避免抖动误触。N、T 可配。
- 捏合点击：检测到捏合进入触发态 → 执行按下 → 释放（一次性点击，不持续按住）。如需拖拽，后续阶段再说，本阶段只做单击。
- 滚轮：进入滚轮模式后，按摆动事件触发，每次滚动行数可配（默认 3 行）。

## 平滑追踪

| 项 | 方案 |
|----|------|
| 滤波 | EMA（指数移动平均），`out = α * raw + (1-α) * prev`，α 默认 0.3（越小越平滑越慢） |
| 预留 | 卡尔曼滤波接口 `ISmoother`，EMA 是默认实现 |
| 增益 | 鼠标位移 = 平滑后位移 × 灵敏度（可配） |
| 死区 | 平滑后位移在死区半径内 → 输出 0（不移动） |
| 中心锚定 | "屏幕正中央停止不做移动"：以画面中心为锚点，手在死区内视为静止 |

> 关键点选择：剑指移动时用**食指指尖(8)与中指指尖(12)的中点**作为指针位置（剑指两指并拢，中点稳定）。

## 坐标映射

- 摄像头归一化坐标 (0~1) → 屏幕像素坐标。
- 镜像翻转：默认开启（前置摄像头自然镜像），可配。
- 映射可加非线性曲线（边缘加速/减速），本阶段先用线性，预留接口。
- 屏幕分辨率通过 `SystemParameters` / Win32 API 获取。

## 鼠标控制（Platform.Mouse）

- `IMouseController`：`MoveTo(x, y)`、`MoveBy(dx, dy)`、`LeftClick()`、`RightClick()`、`Scroll(delta)`。
- 实现 `WindowsSendInputMouseController`：用 `SendInput` (user32.dll) P/Invoke。
- 移动用绝对坐标 `MOUSEEVENTF_ABSOLUTE | MOUSEEVENTF_MOVE`（0~65535 归一化到屏幕）。
- 点击：`MOUSEEVENTF_LEFTDOWN/UP`、`MOUSEEVENTF_RIGHTDOWN/UP`。
- 滚轮：`MOUSEEVENTF_WHEEL`，delta 为 `WHEEL_DELTA` 的倍数。

## 本阶段范围

### 必做

- [x] `Core/Gestures/`：
  - `IGestureRecognizer` + `DefaultGestureRecognizer`：输入 21 关键点 → 输出 `GestureState`（枚举 + 附带数据如指针位置）。
  - 手指伸直/弯曲判定工具方法。
  - 捏合距离计算。
- [x] `Core/Gestures/GestureStateMachine`：状态机（空闲/移动/左键触发/右键触发/滚轮模式），去抖切换。
- [x] `Core/Tracking/`：
  - `ISmoother` 接口 + `EmaSmoother` 实现（α 可配）。
  - `KalmanSmoother` 预留接口（可不实现，仅占位）。
  - `DeadZoneFilter`：死区判定。
  - `CoordinateMapper`：归一化坐标 → 屏幕像素，含镜像翻转。
- [x] `Core/Pipeline/GesturePipeline`：串联 阶段2识别结果 → 平滑 → 死区 → 状态机 → 输出动作事件。
- [x] `Platform/Mouse/`：
  - `IMouseController` 接口 + `WindowsSendInputMouseController` 实现。
  - P/Invoke SendInput 封装。
- [x] `App`：将动作事件接到 `IMouseController`；"启用鼠标仿真"开关生效。
- [x] 调试面板升级：显示当前手势状态、状态机转移、平滑前后坐标、死区命中、鼠标实际位移、滚轮事件计数。
- [x] 预览控件叠加：绘制死区圆圈（屏幕中央）、指针位置十字线、当前手势文本。
- [x] 配置项：灵敏度、平滑系数 α、死区半径、捏合阈值、去抖帧数/时间、滚轮行数、镜像翻转。
### 不做

- 拖拽（持续按住移动）
- 多手 / 惯用手
- 自定义手势
- MediaPipe 实现

## 实现步骤

```
1. 定义 Core/Gestures、Core/Tracking 接口与数据结构
   → 验证：编译通过，Core 无平台引用
2. 实现手指伸直/弯曲判定 + 捏合距离计算（纯函数，可单测）
   → 验证：单元测试覆盖剑指/捏合/张开/空闲四种输入
3. 实现 DefaultGestureRecognizer（21点 → GestureState，无状态判定）
   → 验证：单测给定关键点返回正确手势
4. 实现 GestureStateMachine（带去抖的状态机）
   → 验证：单测状态转移（连续N帧才切换、释放回中）
5. 实现 EmaSmoother + DeadZoneFilter + CoordinateMapper
   → 验证：单测平滑输出、死区归零、坐标映射正确
6. 实现 IMouseController + WindowsSendInputMouseController
   → 验证：手动调用 MoveTo/LeftClick 真实移动鼠标并点击
7. 串联 GesturePipeline（识别→平滑→死区→状态机→动作事件）
   → 验证：剑指移动时鼠标跟随，缓慢平滑
8. 接到 App，启用开关生效
   → 验证：关开关鼠标不动，开开关剑指可控鼠标
9. 调试面板 + 预览叠加升级
   → 验证：状态、坐标、死区、滚轮事件实时可见
10. 参数调优（默认值：α=0.3、死区=0.05、灵敏度=1.5、捏合=0.05、去抖=3帧）
    → 验证：实测移动平滑无抖动，中央死区生效，点击不误触
```
